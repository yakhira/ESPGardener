API endpoints

Home System API

Start pump

Url: /api/water?pump=[pump number]&duration=[max interval for pump work]&active=true

Method: POST

Response body:

{"is_active": "true"}


Stop pump

Url: /api/water?pump=[pump number]&active=false

Method: POST

Response body:

{"is_active": "false"}


Get state of pump

Url: /api/water?pump=[pump number]

Method: GET

Response body:

{"is_active": "false"}
or
{"is_active": "true"}
